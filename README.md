**[Solid Backups](https://solidwp.com/)** is a powerful backup and restore plugin for WordPress designed to help website owners and digital marketers protect their online presence with reliable backup solutions and easy restoration capabilities.

> [!CAUTION]
>
> As of 2025,  the plugin Solid Backup has been **discontinued**. [The plugin is now a "cloud plugin,"](https://solidwp.com/backups/) which means management is done through a web interface we have no access to. However, the latest versions of "Solid Backup Legacy" can still be used.

## Download

[Click here to download Solid Backups](https://gitlab.com/wordpress-premium/solid-backups/-/archive/master/solid-backups-master.zip?ref_type=heads) as a `.zip` file.
